# cl-io-contracts.txt — Producer/consumer contracts for shared LOOP file paths.
#
# FORMAT: SCRIPT  WRITES|READS  relative/path/to/file
# CI check: cl-wire-audit.sh --type-b verifies every WRITES has a READS and vice versa.
#
# When adding a new shared file path:
#   1. Add both WRITES and READS entries here
#   2. Add the path constant to scripts/cl-paths.sh
#   3. Source cl-paths.sh in both scripts (use the variable, not a hardcoded string)
